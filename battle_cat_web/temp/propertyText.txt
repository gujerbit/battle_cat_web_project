공격력 다운 - % 확률로 해당하는 속성의 적의 공격력을 ~초(~F) 동안 % 감소 시킨다
움직임을 멈춘다 - % 확률로 해당하는 속성의 적을 ~초(~F) 동안 정지 시킨다
움직임을 느리게 한다 - % 확률로 해당하는 속성의 적을 ~초(~F) 동안 이동 속도를 1로 고정 시킨다
공격 타겟 한정 - 해당하는 속성의 적만 공격/피해를 줄 수 있다
엄청 강하다 - 해당하는 속성의 적에게 데미지를 50% ~ 40% 만큼만 받고 1.5배 ~ 1.8배의 데미지를 입힌다
맷집이 좋다 - 해당하는 속성의 적에게 데미지를 25% ~ 20% 만큼만 받는다
초 맷집이 좋다 - 해당하는 속성의 적에게 데미지를 16% ~ 14% 만큼만 받는다
초 데미지 - 해당하는 속성의 적에게 3배 ~ 4배의 데미지를 입힌다
극 데미지 - 해당하는 속성의 적에게 5배 ~ 6배의 데미지를 입힌다
날려버린다 - % 확률로 해당하는 속성의 적을 밀친다
워프 - % 확률로 해당하는 속성의 적을 ~초(~F) 후에 ~만큼 (앞 or 뒤)로 이동시킨다
저주 - % 확률로 해당하는 속성의 적의 효과를 ~초(~F) 만큼 봉인시킨다
공격 무효 - % 확률로 해당하는 속성의 적에게 ~초(~F) 동안 데미지를 받지 않는다
공격력 업 - 체력이 % 이하로 떨어지면 공격력이 % 증가한다
살아남는다 - 죽었을 때 % 확률로 체력 1로 살아남는다
성 파괴가 특기 - 성에게 4배의 데미지를 입힌다
크리티컬 - % 확률로 적에게 2배의 데미지를 입힌다 (메탈 능력 무시)
좀비 킬러 - 부활 능력을 가진 적을 처치하였을 때 부활을 막는다
베리어 브레이커 - % 확률로 적의 베리어를 파괴한다
쉴드 브레이커 - % 확률로 적의 쉴드를 파괴한다
혼신의 일격 - % 확률로 적에게 3배의 데미지를 입힌다
격파시 머니 UP - 적을 처치하였을 때 돈을 2배로 획득한다
메탈 - 크리티컬을 제외한 모든 공격을 1의 피해만 받는다
소파동 - % 확률로 레벨 ~의 소파동을 발사한다 (파동의 데미지는 공격력의 20%)
파동 공격 - % 확률로 레벨 ~의 파동을 발사한다
열파 공격 - % 확률로 레벨 ~의 열파를 (시작 지점) ~ (끝 지점) 사이에 랜덤하게 생성한다
공격력 다운 무효 - 공격력 다운 효과를 받지 않는다
움직임을 멈춘다 무효 - 움직임을 멈춘다 효과를 받지 않는다
움직임을 느리게 한다 무효 - 움직임을 느리게 한다 효과를 받지 않는다
날려버린다 무효 - 날려버린다 효과를 받지 않는다
파동 데미지 무효 - 소파동&파동의 데미지를 받지 않는다
열파 데미지 무효 - 열파의 데미지를 받지 않는다
워프 무효 - 워프 효과를 받지 않는다
고대의 저주 무효 - 고대의 저주 효과를 받지 않는다
독 데미지 무효 - 독 데미지를 받지 않는다
파동 스톱퍼 - 소파동&파동의 데미지를 받지 않으며 파동이 자신과 닿으면 파동을 삭제한다
공격력 다운 내성 - 공격력 다운 효과의 시간을 % 감소 시킨다
움직임을 멈춘다 내성 - 움직임을 멈춘다 효과의 시간을 % 감소 시킨다
움직임을 느리게 한다 내성 - 움직임을 느리게 한다 효과의 시간을 % 감소 시킨다
날려버린다 내성 - 밀쳐지는 거리를 % 감소 시킨다
파동 데미지 경감 - 소파동/파동의 데미지를 % 감소 시킨다
열파 데미지 내성 - 열파의 데미지를 % 감소 시킨다
워프 내성 - 워프 당하는 시간을 % 감소 시킨다
고대의 저주 내성 - 고대의 저주 효과의 시간을 % 감소 시킨다
독 공격 데미지 내성 - 독 데미지의 체력 비례 피해량을 % 감소 시킨다
기본 체력 업 - 체력을 % 증가 시킨다
기본 공격 업 - 공격력을 % 증가 시킨다
이동속도 업 - 이동속도를 ~ 증가 시킨다
넉백 횟수 증가 - 넉백 횟수를 ~ 증가 시킨다
생산 코스트 할인 - 생산 비용을 ~원 감소 시킨다
생산 스피드 업 - 생산 시간을 ~초(~F) 감소 시킨다
